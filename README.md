# simple_dorking using python3

===========================================================

Dorking tool using python

Example: python3 dorking.py -s inurl:php?id=12 -e bing -p 3

	  -s = search

	  -e = engine

	  -p = page

===========================================================

Google-dorking

Example:

python google-dork.py -s [search]

python google-dork.py -s [search] -m [max]

	-s		Search
	
	-m		Maximum result
	
	-t		Interval
